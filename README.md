# ‚óè Client Studio
A client portal in Telegram for tracking design projects. Allows clients to see the status, download files and leave comments directly in Telegram.

## Features

- **Authorization by order number** (without passwords)
- **File management** with previews and versions
- **Status tracker** with progress bar
- **File Comments**
- **Deadline calendar**
- **Multi-project** (multiple projects for one client)
- **Responsive design** for Telegram Mobile/Desktop
